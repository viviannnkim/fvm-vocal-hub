

## FVM (From Vivian Music) 보컬 교육 브랜드 홈페이지

체계적인 보컬 교육 브랜드로서의 신뢰감을 전달하고, 5가지 서비스 카테고리로 효과적으로 분기되는 허브형 홈페이지를 제작합니다.

---

### 🎨 디자인 컨셉

**전체 톤**: 딥 네이비 + 골드 포인트의 프리미엄하고 신뢰감 있는 디자인
- 깔끔한 여백과 세련된 타이포그래피
- 프로페셔널한 느낌의 다크/라이트 조화

**키즈 보컬 페이지**: 파스텔 톤과 부드러운 일러스트 요소로 따뜻한 변주
- 밝은 코랄, 민트 등 친근한 색상 활용

---

### 📄 페이지 구조 (총 12개 페이지)

**1. 메인 페이지 (홈)**
- 히어로 섹션: 브랜드 슬로건 + 대표 비주얼
- 브랜드 철학 소개: "대표 설계 커리큘럼, 트레이닝된 강사진"
- 5가지 서비스 카테고리 카드 (아이콘 + 간략 소개)
- 수강 후기 하이라이트
- 카카오톡 상담 CTA 버튼

**2. 서비스 허브 페이지**
- 전체 서비스 5개 카테고리를 한눈에 보여주는 페이지
- 각 서비스별 특징과 대상 안내
- 클릭 시 상세 페이지로 연결

**3. 서비스 상세 페이지 (5개)**
- 🎤 **1:1 프라이빗 보컬레슨**: 맞춤형 커리큘럼, 레슨 진행 방식
- 💻 **온라인 보컬 레슨**: 온라인 장점, 준비물, 진행 방식
- 👥 **그룹 레슨**: 그룹 규모, 장점, 대상
- 🌍 **글로벌 클래스**: 외국인/해외 대상, 영어 설명 강조
- 🌱 **FVM Kids Vocal**: 방문 레슨 시스템, 아이 맞춤 커리큘럼 (따뜻한 디자인 변주)

**4. 커리큘럼 & 시스템 소개**
- 대표 설계 커리큘럼 상세 설명
- 레슨 프로세스 시각화 (단계별 안내)
- 운영 시스템 소개

**5. 강사진 소개**
- 트레이닝된 강사진 프로필 카드
- 각 강사의 전문 분야, 경력 소개

**6. 수강 후기**
- 학생/학부모 후기 카드 형태로 나열
- 카테고리별 필터 기능

**7. 블로그/뉴스**
- 보컬 팁, FVM 소식 등 콘텐츠 공유
- 카드형 목록 + 상세 보기

**8. 문의하기**
- 카카오톡 채널 바로가기 버튼 (메인 CTA)
- 간단한 FAQ 섹션
